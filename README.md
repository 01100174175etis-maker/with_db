# نظام إدارة الحسابات والمبيعات 🏪

**Sales Management System - Enterprise Edition**

نظام متقدم وإحترافي لإدارة الحسابات والمبيعات مع واجهة رسومية حديثة وقاعدة بيانات قوية.

---

## ✨ المميزات الرئيسية

✅ **واجهة رسومية احترافية** - مبنية بـ PyQt5  
✅ **إدارة المبيعات والعملاء** - تتبع كامل لجميع العمليات  
✅ **جلب الأسعار التلقائية** - تذكر آخر سعر لكل منتج  
✅ **قائمة العملاء الجانبية** - عرض سريع وتصفية فورية  
✅ **حساب لحظي للفواتير** - معاينة فورية للإجمالي  
✅ **تصدير Excel متقدم** - تقارير منسقة وجميلة  
✅ **طباعة الفواتير** - توليد فواتير احترافية  
✅ **تعديل وحذف العمليات** - سهولة إدارة السجلات  
✅ **قاعدة بيانات آمنة** - SQLite مع معرفات ذكية  

---

## 🚀 البدء السريع

### المتطلبات
- **Python** 3.8 أو أحدث
- **Windows/Linux/macOS**

### التثبيت على Windows

#### الطريقة 1️⃣ - السهلة (الموصى بها)
```bash
install_windows.bat
```

#### الطريقة 2️⃣ - اليدوية
```bash
pip install -r requirements.txt
```

### التشغيل على Windows

#### الطريقة 1️⃣ - النقر المباشر
انقر على ملف `run.bat` في المجلد

#### الطريقة 2️⃣ - سطر الأوامر
```bash
python main.py
```

### التثبيت على Linux/Mac
```bash
pip install -r requirements.txt
python main.py
```

## 🎯 الاستخدام

### 1️⃣ إضافة عملية بيع جديدة
- أدخل اسم العميل (أو اختره من القائمة)
- اختر المنتج (سيظهر آخر سعر تلقائياً)
- أدخل الكمية والسعر
- اضغط **Enter** أو انقر على **إضافة إلى السجل الحسابي**

### 2️⃣ عرض سريع لحساب العميل
- انقر على اسم العميل في القائمة الجانبية اليسرى
- سيظهر كشف حسابه فوراً في الجدول

### 3️⃣ تعديل عملية
- حدد السطر المراد تعديله
- اضغط **✏️ تعديل السطر**
- عدّل البيانات وانقر **💾 حفظ التعديلات الآن**

### 4️⃣ حذف عملية
- حدد السطر المراد حذفه
- اضغط **🗑️ حذف المحدد**
- أكّد الحذف

### 5️⃣ طباعة فاتورة
- حدد السطر المراد طباعته
- اضغط **🖨️ طباعة الفاتورة**
- ستُنشأ ملف نصي للفاتورة

### 6️⃣ تصدير إلى Excel
- **تصدير عميل واحد**: حدد السطر واضغط **📁 تصدير العميل لإكسل**
- **تصدير جميع العملاء**: اضغط **📊 تحديث السجل الموحد ذو الارتباطات**

## 🔧 التطوير والمساهمة

### بنية قاعدة البيانات

```sql
CREATE TABLE sales (
    id TEXT PRIMARY KEY,           -- معرف ذكي فريد
    customer TEXT NOT NULL,        -- اسم العميل
    item TEXT NOT NULL,            -- اسم المنتج
    qty INTEGER NOT NULL,          -- الكمية
    price REAL NOT NULL,           -- سعر الوحدة
    total REAL NOT NULL,           -- الإجمالي
    sale_date TEXT NOT NULL,       -- تاريخ البيع
    created_at TEXT NOT NULL       -- وقت الإدخال
);
```

### إضافة ميزات جديدة
1. عدّل `main.py` للواجهة الرسومية
2. عدّل `db.py` لتغييرات قاعدة البيانات
3. اختبر البرنامج
4. ارفع التغييرات
